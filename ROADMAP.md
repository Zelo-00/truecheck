# ROADMAP — TrueCheck

Итерации ведутся по `/home/deploy/SkillTask.md` (каскад spec→plan→build→test→review→ship).

## ✅ v0.1 — MVP (готово)
- Разбор текста на эпизоды и ссылки (txt/md/csv/html/rtf/odt/docx/doc/pdf/epub + URL).
- Гибридная привязка источников (файл по содержимому → догрузка из сети).
- Эвристики + строгий ИИ-судья (`AI_RULES.md`, anti-hallucination).
- Отчёт по всем эпизодам + сводный вердикт. FastAPI + nginx + Docker.
- Фронт: минимализм Apple, сплэш, шкала загрузки файла, FAQ + соцсети.

## ✅ v0.2 — UX и доступ к результатам (эта итерация)
- [x] **История проверок** `/history` — список прошлых отчётов (дата, документ, балл, риск).
- [x] **Экспорт отчёта**: скачивание `.md`, печать/сохранение в PDF из браузера.
- [x] **Подсветка ссылок** прямо в тексте эпизода ([1], URL/DOI).
- [x] **Фильтр эпизодов** по статусу (клик по чипу).

## ✅ v0.3 — достоверность, доверие, безопасность (готово)
- [x] **Достоверность:** окно покрывает хвост длинного источника; выбор релевантного источника (не длиннейшего); привязка файлов по порядку загрузки; ИИ-invalid → фолбэк на эвристики.
- [x] **Объяснимость:** человекочитаемое «почему» в каждом эпизоде (отчёт + .md); кнопка «Скопировать ссылку».
- [x] **Безопасность:** rate-limit + опц. токен на `/check`; SSRF-guard на редиректы; история по cookie-сессии; CSP/security-заголовки; non-root контейнер.
- [x] **Наблюдаемость:** реальный лог в `logs/app.log`.
- [x] **Пре-флайт (Фаза 0)** в `SkillTask.md`: тест границ → самооценка 0–100 → стоп → меню.
- Самооценка после фиксов (агенты): корректность 88, безопасность 78, тестируемость 74.

## ⏭ v0.4 — кандидаты (по запросу)
- Асинхронная проверка с прогрессом по эпизодам (job + polling) для длинных документов.
- Экспорт в PDF/DOCX на сервере (weasyprint/python-docx).

- Тесты парсинга реальных PDF/DOCX и сетевого слоя.
